# Functional State Pool

A repeatable check-in to track how you are functioning.

Part of [Dissociative System Witness](https://github.com/FrontSwitch/ds-witness) — a local, private assessment tracker.
DS-Witness handles sampling, tracking, and visualization for this assessment and other datasets.

You can also use the questions and framework on their own, however you prefer.

---

## What

I wanted an assessment I could run frequently (daily or weekly).
Existing tools felt vague or misaligned.
So I tried to build something that would help identify how I was functioning.

**Functional State (FS)** refers to a person's operating capacity.

Like a computer system monitor that shows CPU usage, memory, and active processes, FS tracks how the system is running.
It shifts focus away from pathology and who is fronting.
It asks how the system is performing.

It is the difference between asking "How do you feel?" (subjective/narrative) and "How am I responding?" (objective/mechanical).

**Goal** - A compact, system-level answer to "How am I doing?"

The output helps gauge stability: sustainable or approaching a breakdown.


## How it works

A pool of 36 questions is sampled per run.
Each run presents 12 questions.

Each question is tagged with:

### Mechanism: What is the friction?

**Load** — capacity overwhelmed. Energy, sensory, or cognitive resources are depleted or exceeded. The system is running at or past its limits.

**Resistance** — pushback or avoidance. The system protects autonomy by pushing back against demands, commitments, or engagement — even when that engagement would help.

**Interference** — signal corruption. Something disrupts the normal flow — noise, intrusion, or a loss of agency. The right signal isn't getting through.

### Domain: Where is it showing up?

**Doing** - executive tasks. How the system initiates, sustains, and prioritizes action.
**Feeling** - internal sensations. How the system processes cognitive load, sensory input, and emotional signal.
**Being** - baseline presence. How the system maintains a coherent sense of self across time.
**Relating** - external interactions. How the system manages contact with others and the environment.
